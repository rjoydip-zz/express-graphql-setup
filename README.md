# graphql-examples

Graphql examples.

# Installation

```sh
$ git clone https://github.com/rjoydip/simple-graphql-setup.git
$ cd simple-graphql-setup
$ npm install
```

# Run graphql server

```sh
$ npm start
```

Graphql server will run on [localhost:8000](http://localhost:8000/graphql)
